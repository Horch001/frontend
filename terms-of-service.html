服务条款

最后更新：2024年8月10日

1. 服务概述

Pi 交易平台是一个基于 Pi Network 的数字商品交易平台，为用户提供商品发布、购买、支付和沟通服务。通过使用我们的服务，您同意遵守以下条款和条件。

2. 账户注册和使用

2.1 注册要求

  • 您必须年满18岁或具有完全民事行为能力
  • 您必须拥有有效的 Pi Network 账户
  • 您必须提供真实、准确、完整的注册信息
  • 您有责任保护您的账户安全

2.2 账户责任

  • 您对账户下的所有活动负责
  • 不得将账户转让给他人
  • 发现账户安全问题应立即通知我们
  • 我们有权暂停或终止违规账户

3. 商品发布和销售

3.1 商品要求

  • 商品信息必须真实、准确、完整
  • 不得发布违法、有害、欺诈性商品
  • 不得侵犯他人知识产权
  • 商品图片必须为实物拍摄

3.2 禁止商品

  • 违禁品、危险品、管制物品
  • 假冒伪劣商品
  • 侵犯知识产权的商品
  • 色情、暴力、赌博相关内容
  • 其他违反法律法规的商品

3.3 定价和费用

  • 商品价格由卖家自主设定
  • 平台收取交易手续费（当前为10%）
  • 卖家需要缴纳押金（当前为1000 Pi）
  • 价格变动需提前通知买家

4. 购买和支付

4.1 购买流程

  • 买家确认商品信息和价格
  • 通过 Pi Network 进行支付
  • 支付成功后生成订单
  • 卖家确认订单并发货

4.2 支付安全

  • 所有支付通过 Pi Network 处理
  • 我们不会直接处理您的支付信息
  • 支付失败时资金会原路退回
  • 请确保支付环境安全

5. 订单和配送

5.1 订单处理

  • 卖家应在24小时内确认订单
  • 卖家应在确认后72小时内发货
  • 买家应在收到商品后7天内确认收货
  • 逾期未确认将自动确认收货

5.2 配送责任

  • 卖家负责商品包装和配送
  • 配送费用由双方协商确定
  • 配送风险由卖家承担
  • 买家应提供准确的收货地址

6. 退款和争议

6.1 退款条件

  • 商品与描述严重不符
  • 商品存在质量问题
  • 卖家未按时发货
  • 买家未收到商品

6.2 争议处理

  • 争议应通过平台客服处理
  • 双方应提供相关证据
  • 平台将根据证据做出裁决
  • 裁决结果对双方具有约束力

7. 用户行为规范

7.1 禁止行为

  • 发布虚假信息或欺诈性内容
  • 恶意评价或骚扰其他用户
  • 利用平台漏洞或技术问题
  • 进行洗钱或其他违法活动
  • 干扰平台正常运行

7.2 内容规范

  • 商品描述应客观真实
  • 不得使用误导性语言
  • 图片应清晰展示商品
  • 不得包含不当内容

8. 知识产权

平台内容的知识产权归我们所有，包括但不限于：

  • 网站设计、布局和功能
  • 商标、标识和品牌
  • 软件代码和技术
  • 用户生成内容的使用权

用户保留其发布内容的知识产权，但授予我们使用和展示的权利。

9. 免责声明

重要免责声明：

  • 我们不对用户间的交易承担担保责任
  • 我们不对商品质量、安全性或适用性负责
  • 我们不对因不可抗力导致的服务中断负责
  • 我们不对用户损失承担赔偿责任

10. 服务变更和终止

10.1 服务变更

  • 我们有权随时修改服务内容
  • 重大变更将提前通知用户
  • 继续使用服务视为接受变更

10.2 服务终止

  • 用户可随时停止使用服务
  • 我们可因违规行为终止服务
  • 终止后应处理完未完成交易
  • 账户余额将按规则处理

11. 法律适用和管辖

  • 本条款受中华人民共和国法律管辖
  • 争议应通过友好协商解决
  • 协商不成可提起诉讼
  • 诉讼管辖地为平台所在地

我们可能会不时更新本服务条款。更新后的条款将在网站上发布，继续使用服务即表示您同意新的条款。
